Quick API curl Examples

These commands assume your ANDROID_API_TOKEN is arvindrchauhan1723.

1. Open (create/get today's visit)

curl -s -X POST "https://mvp.dpharma.in/index.php/api/visit/open?token=arvindrchauhan1723" \
  -d "uid=250001"

2. Upload (example image, type=rx)

curl -s -X POST "https://mvp.dpharma.in/index.php/api/visit/upload?token=arvindrchauhan1723" \
  -F "uid=250001" \
  -F "type=rx" \
  -F "file=@/path/to/local/image.png"

3. List visits by date

curl -s "https://mvp.dpharma.in/index.php/api/visit/by-date?token=arvindrchauhan1723&uid=250001&date=2025-08-11"

4. Map Orphans

curl -s -X POST "https://mvp.dpharma.in/index.php/api/visit/map-orphans?token=arvindrchauhan1723"
